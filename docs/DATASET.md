# DASHA-15K Dataset

DASHA-15K (Dashcam Anomaly Scene Hazard Atlas, 15,000 images) is a
benchmark of 15,000 dashcam frames for evaluating vision-language models
on autonomous-driving anomaly detection.

## Composition

| Split | # images |
|---|---|
| Normal driving scenes | 10,000 |
| Anomalous scenes (11 hazard classes) | 5,000 |
| **Total** | **15,000** |

### Anomaly classes (long-tailed)

| Class | Count |
|---|---|
| Oversized or unusual vehicle | 2,037 |
| Human presence anomaly | 728 |
| Strange object on road | 549 |
| Adverse lighting | 415 |
| Animal on road | 402 |
| Infrastructure failure | 251 |
| Fallen debris or vegetation | 229 |
| Vehicle incident | 146 |
| Road surface hazard | 102 |
| Extreme weather | 100 |
| Multi-hazard compound | 33 |

## Annotations (per anomalous image)

Each of the 5,000 anomalous images carries seven attributes:

1. `anomaly_class` тАФ one of the 11 categories above (human-validated).
2. `anomaly_description` тАФ free-form natural language (mean 35 words).
3. `anomaly_key_facts` тАФ list of 3тАУ5 atomic visual observations.
4. `severity` тАФ `minor` / `moderate` / `severe` / `critical`.
5. `ego_risk_level` тАФ `none` / `caution` / `warning` / `danger`.
6. `recommended_action` тАФ one of seven driving actions.
7. `benchmark_difficulty` тАФ `easy` / `medium` / `hard`.

## Annotation pipeline

- **Image curation**: Authors manually inspected every anomalous candidate
  before inclusion. The binary anomaly-vs-normal label is **fully
  human-curated**.
- **Class + description**: Generated by GPT-4o using a structured prompt
  (see `docs/PROMPTS.md`).
- **Cross-validation**: All 5,000 anomalous images re-annotated by Claude
  Sonnet 4.6. Cohen's ╬║ between the two LLM annotators is reported in the
  paper; disagreements were manually adjudicated by an author.

## File layout after download

The Google Drive share contains a single archive `dasha-15k.tar` (~12 GB).
After downloading, extract it into the project's `Data/` directory:

```bash
mkdir -p Data
mv ~/Downloads/dasha-15k.tar Data/
cd Data
tar xf dasha-15k.tar     # creates images/ and dataset.json
rm dasha-15k.tar         # optional: free disk space
cd ..
```

Final on-disk layout (what all scripts in this repo assume):

```
Data/
тФЬтФАтФА images/
тФВ   тФЬтФАтФА data_00001.png

```

## Intended use

DASHA-15K is intended for **evaluation and analysis** of foundation models
on AV anomaly understanding. Authors explicitly **do not** endorse the
deployment of any model evaluated in the paper for unattended autonomous
driving тАФ see Section 8 of the paper for a discussion of safety concerns.
